## TAU - Tuning and Analysis Utilities

![TAU Logo](Images/tau-logo.png)

[See User Docs for more information.](https://docs.rc.fas.harvard.edu/kb/performance-tuning-and-analysis-utilities-tau/)

[TAU (Tuning and Analysis Utilities)](https://www.cs.uoregon.edu/research/tau/home.php) is a comprehensive profiling and tracing toolkit for performance analysis of parallel programs written in Fortran, C, C++, Java, and Python.
