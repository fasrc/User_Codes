### Purpose:

Calculate PI via Monte-Carlo method.

### Contents:

(1) mc_pi.py: Python source code

(2) run.sbatch: Batch-job submission script for sending the job to the queue

### Example Usage:

```bash
sbatch run.sbatch
```
	
### Example Output:

```
Exact value of PI: 3.14159
Estimate of PI: 3.14296
```
